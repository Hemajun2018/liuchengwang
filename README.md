# 流程王项目

一个基于Vue 3、Element Plus和NestJS的流程管理系统。

## 项目结构

- `admin/` - 后台管理系统（Vue 3 + Element Plus）
- `backend/` - 后端服务（NestJS）
- `miniprogram/` - 微信小程序
- `docs/` - 项目文档

## 技术栈

### 后台管理系统
- Vue 3
- Element Plus
- TypeScript
- Vite

### 后端服务
- NestJS
- TypeScript
- MySQL
- TypeORM

### 微信小程序
- 原生微信小程序开发
- TypeScript
